# Support

## How to get help

- Use [GitHub Issues](https://github.com/microsoft/flipcard/issues) for bug reports, feature requests, and questions about the repository.
- Use the included issue templates when possible so maintainers have the context they need.

## Security issues

Please do **not** file public issues for security vulnerabilities. See [SECURITY.md](./SECURITY.md).

## Microsoft Support Policy

This project is maintained in the open and is not covered by standard Microsoft support programs.
