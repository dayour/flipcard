# Icons

These are **placeholder** icons generated by `apps/desktop/scripts/gen-placeholder-icons.mjs`.

Replace them with real branded assets before shipping a release build.

## Regenerate placeholders

From `apps/desktop/`:

```bash
npm run gen:icons
```

This writes:

- `32x32.png`
- `128x128.png`
- `128x128@2x.png` (256×256)
- `icon.png` (1024×1024 source)
- `icon.ico` (multi-size Windows icon)
- `icon.icns` (placeholder; Tauri only requires real `.icns` for macOS bundles)

## Replace with real artwork

Generate from a single 1024×1024 source PNG using Tauri's built-in icon generator:

```bash
npx @tauri-apps/cli icon path/to/flipcard-source.png
```

This produces all platform-specific sizes in this directory.
